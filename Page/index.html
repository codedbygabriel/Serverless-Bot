Serverless Bot

Usually students often needs a discord group chat to create topics, voice channels, study groups etca. This bot aims towards that.

An Agnostic Bot, What?

Although we have a rigid structure, they'll adapt to you server, agnostically.

Commands, a new way of handling study sections!

As your own server grows in size, the number of people that need to create study channels

  • Starter Kit

    If not obvious enough, it's your start point. It creates all necessary Categories and Channels.

    /starter_kit
  • Create Channel

    Starts a input section that allows you to input a Channel Name and how many Students will be allowed to join. Also have a time limit option [defaults to a hour].

    /create_channel [name] [students] (user-time-limit)
  • Create Forum

    Command that allows user to create a forum. Automatically inserts it on Study Channel. /create-forum [name] (guideline)

  • Presentation

    Just a cute hello world, it was my start point for this bot. Maintained for Legacy reasons (xD). /presentation
